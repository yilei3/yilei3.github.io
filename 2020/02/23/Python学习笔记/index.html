列表（list列表是Python中最常用的数据类型

  • 列表中可以容纳不同的数据类型

  • 列表可以通过索引访问元素（和字符串一样索引是从0开始）

  • append() 向列表末尾添加元素

  • pop() 删除列表指定位置的元素，并返回指定位置元素的值，不指定位置则视为删除末尾元素

  • extend() 一次性向列表末尾添加多个元素，对象可以直接为列表

  • insert() 可以将元素插入到列表的指定位置

  • in/not in 使用in或者not in可以判断一个元素在不在列表中，返回值为True/False

  • remove() remove()用于按值删除元素，找到第一个符合的元素然后删除

  • del 用于删除指定列表中的元素

  • sum()函数 用于对列表中的元素进行求和，但是列表中的元素必须都是数值类型的
